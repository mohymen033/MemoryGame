Play Instruction:
1. Use Tab to select navigate to cards board or Game reset element.

2. Use left,right,up,down to navigate card element of the borad.

3. Use enter to flip selected card element of the board.

4. Use enter to fire restart game button event.

5. For every correct match 1 point and every wrong match -1 point.
6. For new session point and time is reset.
